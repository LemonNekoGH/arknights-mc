# arknights-mc

明日"方"舟

### 这是什么？

    这位朋友，明日方舟您知道吗？
    那我的世界您知道吗？
    都知道是吧，那在我的世界里玩明日方舟呢？

明日方舟和Minecraft都是我们喜爱的游戏，我们正想办法让您在一个游戏里体验到两个游戏的元素，于是我们决定制作此MOD。

请注意，本项目不属于Minecraft或明日方舟的附属项目，只是游戏爱好者为了爱好而创建的项目。
